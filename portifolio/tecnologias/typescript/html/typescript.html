Typescript

TypeScript é um superconjunto de JavaScript desenvolvido pela Microsoft que adiciona tipos estáticos opcionais ao JavaScript. Ele é projetado para ser compatível com o JavaScript existente, ou seja, qualquer código JavaScript válido também é código TypeScript válido

3 Prós do Typescript:

  • Suporte a Recursos Modernos
  • Facilidade de Manutenção
  • Tipagem Estática

3 Limitações do Typescript:

  • Curva de Aprendizado
  • Necessidade de Compilação
  • Complexidade de Configuração
