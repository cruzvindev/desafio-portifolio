Java

Java é uma linguagem de programação orientada a objetos e uma plataforma de desenvolvimento amplamente utilizada, lançada pela primeira vez pela Sun Microsystems em 1995 e atualmente mantida pela Oracle Corporation. Java é conhecido por sua portabilidade, segurança e robustez.

3 Prós do Java:

  • Portabilidade
  • Segurança
  • Variedade de Bibliotecas

3 Limitações do Java:

  • Desempenho
  • Consumo de Memória
  • Complexidade da JVM

Grandes Projetos que Usam Java:
