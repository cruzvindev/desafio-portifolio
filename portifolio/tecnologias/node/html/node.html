Node

Node.js é um ambiente de execução JavaScript construído sobre o motor V8 do Google Chrome. Ele permite que você execute código JavaScript no lado do servidor, o que antes só era possível no navegador.

3 Prós do Node JS:

  • Modelo de I/O Não Bloqueante
  • Ecossistema NPM
  • Escalabilidade

3 Limitações do Node JS:

  • Compatibilidade com Módulos
  • Documentação Não Uniforme
  • Problemas de Debugging
