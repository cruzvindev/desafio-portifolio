Postgre SQL

O PostgreSQL é um sistema de gerenciamento de banco de dados relacional de código aberto, altamente avançado e robusto. Ele é conhecido por sua conformidade com os padrões SQL e por oferecer um rico conjunto de funcionalidades

3 Prós do Postgre:

  • Conformidade com Padrões e Funcionalidades Avançadas
  • Extensibilidade
  • Comunidade Ativa

3 Limitações do Postgre:

  • Desempenho em Consultas Simples com Grande Volume de Dados
  • Suporte a Alguns Recursos Avançados de Nuvem
  • Recursos de Administração e Ferramentas de Terceiros
