HTML 5

HTML (Hypertext Markup Language) é a linguagem padrão para criação e estruturação de páginas web. É uma linguagem de marcação que utiliza tags para definir o conteúdo e a estrutura de uma página web.

3 Prós do HTML 5:

  • Simplicidade
  • Padrão da Web
  • Facilidade de Manutenção

3 Limitações do HTML:

  • Compatibilidade Entre Navegadores
  • Dependência de Outras Tecnologias
  • Limitações de Funcionalidades

Grandes Projetos que Usam HTML:

  • Google
  • Facebook
  • Amazon
  • Twitter
  • Wikipedia
  • YouTube
